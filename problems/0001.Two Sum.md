# 1.Two Sum
[0001.Two Sum](https://leetcode.com/problems/two-sum/)

## 解題思路:

### Brute-force:

先記錄一個值temp,再找與目標target的差值.  
避免內層for循環,每次在多運算一次.

```c
int* twoSum(int* nums, int numsSize, int target, int* returnSize){
    *returnSize = 2;
    int *ret = malloc(2 * sizeof(int));
    for (int i = 0; i < numsSize; i++){
        int temp = target - nums[i];
        for (int j = i + 1; j < numsSize; j++){
            if (temp == nums[j]){
                ret[0] = i;
                ret[1] = j;
                return ret;
            }
        }
    }
    return ret;
}
```

### Hash:

本題到哈希的概念,在C中需要先引入下面C的library.  
<uthash.h>沒記錯的話,是由民間大老自己造的輪子.

```c
#include<uthash.h>
```

```c
struct hashTable{
    int key;
    int val;
    UT_hash_handle hh;
};
struct hashTable *hash = NULL;

struct hashTable* find( int key){
    struct hashTable *temp = NULL;
    HASH_FIND_INT(hash, &key, temp);
    return temp;
}

void insert( int key, int val){
    struct hashTable *temp = find( key);
    if ( !temp){
        struct hashTable *newNode = malloc(sizeof(struct hashTable));
        newNode->key = key;
        newNode->val = val;
        HASH_ADD_INT(hash, key, newNode);
    }
    else {
        temp->val = val;
    }
}

int* twoSum(int* nums, int numsSize, int target, int* returnSize){
    hash = NULL;
    for ( int i = 0; i < numsSize; i++){
        struct hashTable *temp = find( target - nums[i]);

            return ans;
        }
        insert( nums[i], i);
    }

    return NULL;
}
```
